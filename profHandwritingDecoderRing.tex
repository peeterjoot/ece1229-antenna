I found Prof. Eleftheriades' handwriting tricky to decode in a number of cases.  Here's a handy dandy codex should anybody else have the same troubles
\begin{itemize}
\item [\color{DarkOliveGreen}{\textbf{(a)}}]
Greek letter \(\Omega\) is written as a circle floating above a face up square bracket.
%, as in \cref{fig:greekStyleOmega:greekStyleOmegaFig1}.
%\imageFigure{../figures/ece1229-antenna/greekStyleOmegaFig1}{Prof. Eleftheriades' \( \Omega \).}{fig:greekStyleOmega:greekStyleOmegaFig1}{0.05}
\item [\color{DarkOliveGreen}{\textbf{(b)}}]
Greek letter \( \sigma \) is written like a number 6, slightly tipped.
%, as in \cref{fig:sigma:sigmaFig1}.
%\imageFigure{../figures/ece1229-antenna/sigmaFig1}{Prof. Eleftheriades' \( \sigma \).}{fig:sigma:sigmaFig1}{0.05}
\item [\color{DarkOliveGreen}{\textbf{(c)}}]
J looks like pi with a tail.
% \cref{fig:profJ:profJFig1}.
%\imageFigure{../figures/ece1229-antenna/profJFig1}{Prof Eleftheriades' J.}{fig:profJ:profJFig1}{0.05}
\item [\color{DarkOliveGreen}{\textbf{(d)}}]
Greek letter \( \lambda\) looks like
%\cref{fig:profLambda:profLambdaFig2}
a mirror image of his 'h'.
%\imageFigure{../figures/ece1229-antenna/profLambdaFig2}{Prof. Eleftheriades' \( \lambda \).}{fig:profLambda:profLambdaFig2}{0.05}
\item [\color{DarkOliveGreen}{\textbf{(e)}}]
Greek letter \( \mu\) can look like an M.
% \cref{fig:profMu:profMuFig3}.
%\imageFigure{../figures/ece1229-antenna/profMuFig3}{Prof. Eleftheriades' \( \mu \).}{fig:profMu:profMuFig3}{0.05}
\end{itemize}
